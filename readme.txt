总体思路：
   首先在做之前看了下豆瓣，没有反爬虫策略，所以只管下载网页，下载网页前要获取url，获取时，在浏览器上先按评价排序，然后复制上面url即可，
翻页也是通过构造url解决的，然后就是用正则式从下载的网页里面提取出想要的数据，最后再用相关的库将数据存到excel中。
    对于多线程的使用应该注意，一般情况下对于线程的数目，此数值越大速度越快，但是越可能被反爬虫发现，此时限制的线程个数为20。程序的运行时间大约为两个小时。

   程序里面有一个正则的方法进行匹配想要的数据、一个线程的方法，还有一个线程的测试方法。首先是线程每执行20次的时候就将数据放在本地的d盘上面的
一个文件里，等到全部爬完之后再从里面去取。
